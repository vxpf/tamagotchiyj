Tamagotchi YJ C# Project

Over dit project
Dit project is een virtuele Tamagotchi gemaakt in C#. Het doel van het project is een digitale pet bouwen waarbij de speler moet zorgen voor het dier. Het project helpt bij het leren van game logica, object oriented programming en software ontwerp. Veel Tamagotchi projecten werken met een digitaal huisdier dat verzorging nodig heeft zoals eten en slapen.

Project informatie
Naam project: TamagotchiYJ
Programmeertaal: C#
Type project: Schoolproject

Game concept
De speler krijgt een virtueel huisdier. Het huisdier heeft status waarden zoals honger, energie en gezondheid. De speler moet acties uitvoeren om het dier in leven te houden.
